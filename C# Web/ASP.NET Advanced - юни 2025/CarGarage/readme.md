# 🚗 Auto Service Management System

## 📌 Overview

A full-stack software solution for managing automotive service operations. The system helps service providers efficiently track repairs, manage used parts, and generate invoices for both individual and corporate clients.

Additionally, the platform features an integrated marketplace that enables users to buy and sell vehicles and automotive parts, creating a unified ecosystem for automotive businesses.

---

## ⚙️ Core Features

### 🔧 Service Management

* Track and manage all performed repairs
* Maintain detailed service history for each vehicle
* Organize workflows within the service

### 📦 Parts Management

* Record parts used in repairs
* Track availability and usage
* Improve cost control and transparency

### 🧾 Invoicing

* Generate invoices for:

  * Individual clients
  * Corporate clients
* Automatic price calculations
* Structured financial tracking

### 🛒 Marketplace

* Buy and sell vehicles
* Trade automotive parts
* Interaction between different service users

---

## 🛠️ Technologies Used

* **C# (.NET)**
* **Microsoft SQL Server (MSSQL)**
* **JavaScript**
* HTML & CSS

---

## 🏗️ Architecture

The application follows a layered architecture:

* Backend (C# / .NET) – business logic and data processing
* Database (MSSQL) – structured data storage
* Frontend (JavaScript, HTML, CSS) – user interface

---

## 🎯 Purpose

The goal of this project is to digitalize automotive service management
by combining repair tracking, inventory management,
invoicing, and a marketplace into a single platform.

---

## ▶️ How to Run

1. Clone the repository
2. Set up the MSSQL database
3. Configure the connection string
4. Run the application through Visual Studio
5. Access via browser
